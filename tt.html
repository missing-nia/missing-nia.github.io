Twisted Truths

Video Demonstration

Twisted Truths is a fan project I have been providing development work for as the primary programmer. Work on the project is currently on hold, but overall aims to recreate the feeling of the DanganRonpa game series. Currently, I have done extensive work on the project, developing all systems, gameplay mechanics, UI implementations, and more in Unity.

Throughtout its development, I have implemented many systems, from dynamic UI elements to object interaction systems. For one of these system features, the scene transitions, I have made a short presentation detailing the design process and how it works here.
